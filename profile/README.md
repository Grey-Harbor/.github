Grey Harbor builds software with a long horizon in mind.

This organization serves as both a workshop and a technical journal — collecting experiments, tooling, infrastructure patterns, application architecture research, and projects that help shape the direction of the studio.

Areas of focus include:

* Flutter and React Native application architecture
* TypeScript services and developer tooling
* AI-enabled and local AI workflows
* Serverless and event-driven infrastructure
* Local-first software
* Infrastructure automation and edge computing systems

We care deeply about maintainability, operational clarity, and software that remains useful long after the initial release cycle.
